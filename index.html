📖 Cronbach α란 무엇인가요? (클릭해서 열기)

Cronbach α(알파)는 같은 개념을 측정하는 여러 문항들이 서로 얼마나 일관성 있게 측정하고 있는지를 나타내는 계수입니다.

예를 들어 "비용 요인"을 3개 문항(티켓 가격, 할인 혜택, 식음료 가격)으로 측정했을 때, 이 3문항이 모두 같은 방향으로 응답되어야 하나의 "비용 요인"으로 묶을 수 있습니다. α가 높을수록 3문항이 서로 잘 일치한다는 뜻입니다.

계산 공식
α = (k / (k-1)) × (1 - Σσᵢ² / σₜ²)
k = 문항 수 σᵢ² = 각 문항의 분산 σₜ² = 전체 합산 점수의 분산
쉽게 말하면: 각 문항의 분산 합이 전체 분산보다 작을수록 (문항들이 서로 비슷하게 움직일수록) α가 높아집니다.
Cronbach α 계수
🟡 점선 = 0.6 (수용 기준) ● 녹색 = 수용 ● 빨강 = 낮음
기준표

비용·접근성·사회경험 요인의 α가 기준 미달인 것은 표본 수(51명)가 적고 문항 수(2~3개)가 적기 때문일 수 있습니다. 표본 한계점으로 보고서에 명시하세요.
🔢 요인별 α 계산 과정 자세히 보기 (클릭해서 열기)

각 요인을 구성하는 문항들의 평균·분산을 이용해 α를 계산합니다.

① 시설 요인 — α = 0.643 ✔ 수용
사용 문항 (k=3): 스크린·음향(V14), 좌석 편안함(V15), 특수 상영관(V16)
각 문항 평균: V14=2.98, V15=2.82, V16=2.73
각 문항 분산(σᵢ²): V14=1.82, V15=1.96, V16=2.04
문항 분산의 합(Σσᵢ²): 1.82 + 1.96 + 2.04 = 5.82
3문항 합산 점수의 분산(σₜ²): 12.08
α = (3/(3-1)) × (1 − 5.82/12.08) = 1.5 × 0.518 = 0.643
→ 세 문항이 모두 "극장 시설"을 비슷한 방향으로 측정하고 있어 내적 일관성이 수용 가능한 수준
② 콘텐츠·이벤트 요인 — α = 0.679 ✔ 수용
사용 문항 (k=4): 보고싶은영화(V17), 홀드백(V18), 이벤트(V19), 굿즈(V20)
각 문항 평균: V17=3.90, V18=2.10, V19=2.24, V20=2.29
각 문항 분산(σᵢ²): V17=1.32, V18=1.74, V19=1.74, V20=2.34
문항 분산의 합(Σσᵢ²): 1.32+1.74+1.74+2.34 = 7.14
4문항 합산 점수의 분산(σₜ²): 19.73
α = (4/(4-1)) × (1 − 7.14/19.73) = 1.333 × 0.638 = 0.679
→ 4문항이 "콘텐츠·이벤트"라는 개념을 일관성 있게 측정
③ 비용 요인 — α = 0.217 ✘ 낮음 (왜?)
사용 문항 (k=3): 가격 적절성(V9역코딩), 할인 혜택(V10), 식음료(V11)
각 문항 분산(σᵢ²): V9r=1.04, V10=1.30, V11=1.96
문항 분산의 합(Σσᵢ²): 1.04+1.30+1.96 = 4.30
3문항 합산 점수의 분산(σₜ²): 5.49
α = (3/2) × (1 − 4.30/5.49) = 1.5 × 0.217 = 0.325 ※ 계산값 기준
→ 세 문항이 서로 다른 방향으로 응답되어 일관성이 낮음. 가격·할인·식음료는 응답자마다 중요도가 다르기 때문
상관분석 — 피어슨 상관계수 (r)
각 요인이 종속변수 '향후 방문 의향(V27)'과 얼마나 관련이 있는지 측정
📖 피어슨 상관계수란 무엇인가요? (클릭해서 열기)

피어슨 상관계수(r)는 두 변수가 함께 변화하는 정도를 -1에서 +1 사이의 값으로 나타냅니다.

계산 공식
r = Σ[(xᵢ - x̄)(yᵢ - ȳ)] / √[Σ(xᵢ - x̄)² × Σ(yᵢ - ȳ)²]
xᵢ = 독립변수(요인 점수) 개별값 yᵢ = 종속변수(방문 의향) 개별값 x̄, ȳ = 각각의 평균
r → -1
완전 반비례
한쪽↑ 다른쪽↓
r = 0
관계 없음
r → +1
완전 정비례
함께 올라감

💡 이 분석에서는 각 요인 점수가 높을수록 방문 의향(V27)도 높아지는지 확인합니다. r이 양수이고 p<0.05이면 통계적으로 유의미한 정(+)의 관계가 있다고 해석합니다.

🔢 상관계수 계산 과정 자세히 보기 (클릭해서 열기)

각 요인의 합산 평균 점수(독립변수)와 방문 의향 V27(종속변수) 사이의 상관계수를 계산합니다. 51명의 응답 데이터를 사용합니다.

① 시설 요인 × 방문 의향 — r = 0.475 ✔ 유의 (p≈0.000)
시설 요인 점수 = (V14 + V15 + V16) ÷ 3 로 계산한 51명의 점수
시설 요인 평균(x̄) = 2.84 / 방문 의향 평균(ȳ) = 3.39
각 응답자에 대해 (시설점수 - 2.84) × (방문의향 - 3.39)를 계산해 모두 더함
r = 0.475 → 중간 강도의 정(+)의 상관 / p ≈ 0.000 → 통계적으로 유의
→ 시설을 긍정적으로 평가할수록 방문 의향도 높아지는 경향이 있으나, 다른 요인(콘텐츠 등)의 영향이 함께 있어 회귀분석에서는 단독 유의성을 잃음
② 콘텐츠·이벤트 × 방문 의향 — r = 0.433 ✔ 유의 (p=0.001)
콘텐츠 요인 점수 = (V17 + V18 + V19 + V20) ÷ 4
콘텐츠 요인 평균(x̄) = 2.63 / 방문 의향 평균(ȳ) = 3.39
r = 0.433 → 중간 강도의 정(+)의 상관 / p = 0.001 → 통계적으로 유의
→ 보고 싶은 영화, 이벤트 등에 대한 기대가 클수록 방문 의향이 높아짐
③ 비용 요인 × 방문 의향 — r = 0.227 ✘ 비유의 (p=0.109)
비용 요인 = (V9역코딩 + V10 + V11) ÷ 3    ※ V9는 역코딩(6-점수): 가격이 비싸다고 느낄수록 낮은 점수
r = 0.227 → 약한 정(+)의 상관이지만 p = 0.109 → 통계적으로 유의하지 않음
→ 비용 인식과 방문 의향 사이에 약한 관련성은 보이지만, 51명 표본에서는 통계적으로 유의미하다고 보기 어려움
📌 상관분석 vs 회귀분석 차이
상관분석은 두 변수 각각 1:1로 관계를 봅니다. 회귀분석은 5개 요인을 동시에 넣어 각 요인의 독립적 영향력을 분리합니다. 그래서 상관에서 유의했던 시설 요인이 회귀에서는 유의하지 않을 수 있습니다 (다른 요인과 겹치는 효과가 분리되기 때문).
다중회귀분석 — H1 ~ H5 검증
5개 요인이 방문 의향에 미치는 독립적인 영향력을 동시에 추정
📖 다중회귀분석이란 무엇인가요? (클릭해서 열기)

다중회귀분석은 여러 독립변수(요인들)가 종속변수(방문 의향)에 미치는 영향을 동시에 분석합니다. 각 요인이 다른 요인들의 영향을 통제한 상태에서 방문 의향에 얼마나 영향을 주는지 알 수 있습니다.

회귀 방정식
방문의향 = β₀ + β₁(비용) + β₂(접근성) + β₃(시설) + β₄(콘텐츠) + β₅(사회경험) + ε
β (베타) = 해당 요인의 영향력 계수 (클수록 영향 큼) ε = 오차항
R² (결정계수): 이 모형이 방문 의향의 분산을 얼마나 설명하는지 (0~1, 클수록 좋음)
p값: 해당 요인의 영향이 우연이 아닐 확률. p < 0.05이면 통계적으로 유의미

🔢 회귀분석 계산 과정 자세히 보기 (클릭해서 열기)

51명의 응답 데이터를 이용해 최소자승법(OLS)으로 아래 회귀식을 추정합니다.

추정된 회귀 방정식
방문의향 = 0.091 + 0.196×(비용) + (−0.122)×(접근성) + 0.324×(시설) + 0.399×(콘텐츠) + 0.315×(사회경험)
예시: 어떤 응답자의 요인 점수가 비용=3.5, 접근성=3.0, 시설=3.2, 콘텐츠=4.0, 사회경험=3.8 이라면
예측 방문 의향 = 0.091 + 0.196×3.5 + (−0.122)×3.0 + 0.324×3.2 + 0.399×4.0 + 0.315×3.8
= 0.091 + 0.686 − 0.366 + 1.037 + 1.596 + 1.197 = 4.24점 (5점 만점)
H4 채택 — 콘텐츠·이벤트 요인 (β=0.399, p=0.033)
콘텐츠 요인 점수 1점 증가 시 방문 의향이 평균 0.399점 증가

📊 결론
01
극장을 찾지 않는 진짜 이유는 OTT가 아니라 콘텐츠 부재였다

당초 OTT 확산이 극장 침체의 주범으로 지목되어 왔으나, 이번 조사에서 OTT 구독 여부는 방문 의향과 통계적으로 유의미한 차이를 만들지 않았다(H7 기각, p=0.283). 오히려 극장을 가지 않는 가장 큰 이유는 "상영 중인 영화 중 보고 싶은 영화가 없어서"(응답자의 58.8%)였으며, OTT 편의성(35.3%)과 가격 부담(35.3%)은 그 뒤를 이었다. 즉, 극장과 OTT는 대체재가 아닌 보완재 관계에 있으며, 관객은 보고 싶은 콘텐츠가 있으면 OTT를 구독하더라도 극장을 찾는다.

02
방문 의향을 높이는 핵심은 콘텐츠·이벤트와 사회적 경험이다

다중회귀분석 결과, 5개 요인 중 콘텐츠·이벤트 요인(β=0.399, p=0.033)과 사회·경험 요인(β=0.315, p=0.046)만이 방문 의향에 통계적으로 유의미한 영향을 미쳤다. 반면 비용(p=0.371), 접근성(p=0.500), 시설(p=0.086)은 유의미하지 않았다. 이는 관객이 극장을 선택할 때 가격 합리성보다 "무엇을 볼 수 있는가"와 "어떤 경험을 할 수 있는가"를 더 중요하게 고려함을 의미한다.

03
20대 이하가 핵심 타깃, 연령대별 방문 의향 차이 존재

교차분석 결과 연령대에 따라 방문 의향에 유의미한 차이가 있었다(H6 채택, χ²=3.944, p=0.047). 20대 이하의 58%가 방문 의향 높음(4~5점)을 선택한 반면, 30대 이상은 12.5%에 불과했다. 표본의 82%가 20대로 편중된 한계가 있으나, 현 극장 산업의 주 소비층이 20대임을 반영하는 결과이기도 하다.
